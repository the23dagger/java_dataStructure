## 💻 Nasıl Kullanılır?

### 1. Repo’yu klonla

```bash
git clone https://github.com/TheAlgorithms/Java.git
cd Java
```

### 2. Çalıştır

```bash
mvn clean install
```

### 3. Kodları incele

```bash
src/main/java/com/thealgorithms/
```
